# TUGAS-TAG-HTML
``html``
1. html
Fungsi:
Sebagai struktur utama (kerangka) dari sebuah halaman web. Semua elemen HTML harus berada di dalam tag ini supaya browser bisa membaca halaman dengan benar.
Contoh:
Pada kode tadi, seluruh isi dari mulai judul sampai teks berada di dalam tag html, artinya semua bagian website dibungkus di dalamnya.
2. headFungsi:
Menyimpan *informasi penting tentang halaman* yang tidak ditampilkan langsung di layar, seperti judul halaman atau pengaturan lainnya.
Contoh:
Tulisan “Website Pribadi” muncul di tab browser karena diletakkan di dalam bagian head.
 3. body
Fungsi:
Menampilkan semua isi yang bisa dilihat oleh pengguna, seperti teks, gambar, dan link.
Contoh:
Bagian “Profil Saya”, paragraf, gambar, dan daftar hobi semuanya ditampilkan karena berada di dalam body.
4. h1 – h6
Fungsi:
Digunakan untuk membuat *judul dan subjudul* agar halaman lebih terstruktur dan mudah dibaca.
Contoh:
“Profil Saya” sebagai judul utama (h1), lalu “Pendidikan” dan “Hobi” sebagai subjudul.
5. Fungsi:
Digunakan untuk membuat paragraf teks supaya tulisan rapi dan tidak berantakan.
Contoh:
Kalimat “Saya adalah siswa yang sedang belajar membuat website menggunakan HTML” ditampilkan sebagai paragraf.

6. a
Fungsi:
Digunakan untuk membuat link/tautan yang bisa diklik.
Contoh:
Tulisan “Instagram saya” bisa diklik dan akan membuka halaman Instagram.

7. img
Fungsi:
Digunakan untuk menampilkan gambar di halaman web.
Contoh:
Gambar “foto.jpg” ditampilkan sebagai foto profil di halaman.

8. ul
Fungsi:
Digunakan untuk membuat daftar tidak berurutan (bullet/titik).
Contoh:
Daftar hobi ditampilkan dalam bentuk titik-titik seperti
* nonton 
* mendengarkan musik


9. li
Fungsi:
Digunakan untuk membuat isi dari daftar.
Contoh:
'Nonton', dan 'mendengarkan musik' adalah item dalam daftar.

10. br
Fungsi:
Digunakan untuk membuat baris baru tanpa membuat paragraf baru.
Contoh:
Alamat ditampilkan ke bawah per baris (jalan, nomor, negara) menggunakan br.

11. div
Fungsi:
Digunakan untuk mengelompokkan beberapa elemen dalam satu bagian agar lebih rapi dan mudah diatur.
Contoh:
Bagian “Informasi Penting” dibungkus dalam div supaya jadi satu kelompok.

12. b / strong
Fungsi:
Digunakan untuk membuat teks menjadi tebal agar lebih menonjol.
Contoh:
Kata “PERHATIAN” dibuat tebal supaya terlihat penting. 

 13. i
Fungsi:
Digunakan untuk membuat teks menjadi miring.
Contoh:
Kalimat “Learning by doing” dibuat miring untuk penekanan atau gaya penulisan.
